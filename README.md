# Recommender Systems Projects

This repository contains two recommendation-system projects developed using the MovieLens 1M dataset:

- **Neural Collaborative Filtering (NCF):** learns user–item interactions by combining matrix factorization and a multilayer perceptron.
- **BERT4Rec:** uses a Transformer-based architecture to model user interaction sequences and generate recommendations.

Both projects include data preprocessing, model training, evaluation with Recall@10 and NDCG@10, and experiment results.

## Projects

- [`Group25_A1_NCF`](Group25_A1_NCF/) — Neural Collaborative Filtering
- [`Group25_A2_BERT4Rec`](Group25_A2_BERT4Rec/) — Sequential recommendation with BERT4Rec

See each project's README for installation instructions, configuration details, and usage.
